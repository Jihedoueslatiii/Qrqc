📌 À propos

QRQC (Quick Response Quality Control) est une plateforme de gestion de qualité et de production permettant :

  • La gestion des produits et leurs KPI (Coût, Délai, Qualité)
  • Le suivi des projets et leurs évolutions
  • La visualisation à travers un dashboard dynamique
  • L’analyse des causes d’anomalies
  • La création de plans d’action adaptés

🧭 Navigation

Utilisez la barre latérale pour accéder aux modules suivants :

  • Dashboard : Vue globale sur les performances
  • Produits : Ajouter / Modifier un produit
  • KPI : Saisir les données par KPI (Coût, Délai, Qualité)
  • Analyse : Analyser les causes
  • Plan d’action : Définir les réponses correctives

➕ Ajout de données

  • Cliquer sur ➕ Nouveau KPI pour créer un nouvel indicateur avec son pilote et objectif
  • Ajouter des lignes avec le bouton ➕ par KPI
  • Utiliser le bouton ➖ Ligne d’arrêt pour noter une interruption de production

📊 Lecture des indicateurs

Chaque ligne de KPI affiche :

  • 📅 Date de la mesure
  • 🔢 Valeurs déclarées ou mesurées
  • ✅ Résultat calculé automatiquement (ex. : % pièces conformes)
  • 🎯 Objectif à atteindre
  • 👤 Nom du pilote

📤 Exportation

Utilisez le bouton Exporter Excel pour télécharger un rapport personnalisé de vos données KPI selon les filtres actifs.

🔍 Analyse des causes

Dans la section Analyse, vous pouvez :

  • Associer une ou plusieurs causes à un KPI anormal
  • Choisir le type de problème (qualité, temps, coût, etc.)

📝 Plans d’action

Pour chaque cause détectée :

  • Créer un plan d’action correctif
  • Assigner un responsable
  • Suivre l’état d’avancement (à faire, en cours, terminé)

📈 Dashboard

La page de tableau de bord vous permet de :

  • Visualiser les performances globales
  • Comparer les résultats entre produits / projets
  • Suivre les progrès à travers des graphiques et camemberts
